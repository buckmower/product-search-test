Project Definiation:

Build a small product discovery page that lets someone search and browse these home goods. 
Keep it small: a page where you can search, with results that feel amazing and helpful. We don't need to build the whole store.

We care more about the decisions you make around the search than about the search itself. 
Open the data, get a feel for what it's really like, and build the version you'd be happy to put in front of real people
